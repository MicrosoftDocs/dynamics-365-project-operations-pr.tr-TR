# <a name="create-product-catalog-items-project-service"></a>Ürün kataloğu maddeleri oluşturma (Project Service)

[!INCLUDE[cc-applies-to-psa-app-1x-2x](../includes/cc-applies-to-psa-app-1x-2x.md)]
