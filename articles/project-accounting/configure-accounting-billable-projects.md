# <a name="configure-accounting-for-billable-projects"></a><span data-ttu-id="259ef-103">Faturalanabilir projeler için muhasebeyi yapılandırma</span><span class="sxs-lookup"><span data-stu-id="259ef-103">Configure accounting for billable projects</span></span>

<span data-ttu-id="259ef-104">_**Şunlar için geçerlidir:** Kaynağı/stoğu tutulmayanları temel alan senaryolar için Project Operations, Lite dağıtımı-proforma faturalamayı yönetme_</span><span class="sxs-lookup"><span data-stu-id="259ef-104">_**Applies To:** Project Operations for resource/non-stocked based scenarios, Lite deployment - deal to proforma invoicing_</span></span>
